    Library for microcontroller AVR 8bits

    Project maintained by antoine163 Hosted on GitHub Pages — Theme by mattgraham

    Bienvenue sur la page GitHub de avr_swift.

    avr_swift est une bibliothèque réaliser dans le but de développer plus rapidement le software des microcontrollers de type AVR (8bit mega et tiny) de chez ATMEL. Elle contiens, les fichiers pour la gestion des coupleurs ainsi que des driver s'appuient sur ces coupleurs. La compatibilité de avr_swite avec tout les avr 8bit ne peut être garantie, même si c'est l'un de c'est but.

    Installation.

    Il faut déjà que vous récupérer les source du projet.Soit directement avec git : 

    $ git clone git://github.com/antoine163/avr_swift.git
    Ou en téléchargent l'archive du dernier commit.

    Utilisation.

    Un exemple de makefile et donnent dans le dossier template. Pour une tout autre configuration il suffi d'inclure peripherals et drivers dans votre projet et les fichiers .c que vous utiliser.

    Le makefile fournie dans template permet de compiler en release ou debug, de programmer le composent cible et de générer la documentation de votre projet.

    Documentation (Doxygen).

    Les fichiers constituent avr_swift ont été conçus de sort à générai de la documentation automatique grâce au logicielle doxygen. Vous pouvez générai la documentation en faisant "make" dans le dossier avr_swift ou un "make clean" pour supprimer la doc. Vous pouvez toutefois la consulter en linge.

    Précision/autre.

    Une bibliothèque développer par ATMEL, nommer ASF "Atmel Software Framework" est disponible elle est en développement et supporte un nombre limiter de coupleurs/fonctionnalité pour AVR mega. Donc à voir pour le maintien futur d'avr_swift.
