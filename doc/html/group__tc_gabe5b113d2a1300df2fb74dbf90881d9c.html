enum tc_mode_e

Les différents modes.

On peut distinguer deux grande famille de valeur. Les timer 0 et 2 sur 8 bit et les timer 1, 3, 4, 5 sur 16 bit. Pour plus de renseignement, veiller consulter la documentation de votre composent. A utiliser avec tc_mode_select().

Note
Les dénomination OCn (timer ayant qu'un seul comparateur) et OCnX, représente les comparateurs, avec n représentent le numéro du timer et X représentent le nom des comparateurs.

5 type de mode possible :

Normale.
C'est le mode qui est configurée par défaut et le plus basique. Le(s) registre(s) de comparaison (OCn/OCnX) sont mis à jour immédiatement. Le débordement du conteur un ci que le drapeau de débordement s'effectu au maximums (soit 0xff pour un timer de 8 bit ou 0xffff pour un timer de 16 bit).
CTC (Clear Timer on Compare).
Le(s) registre(s) de comparaison (OCn/OCnX) sont mis à jour immédiatement. Le débordement du conteur un ci que le drapeau de débordement s'effectu sur le registre de comparaison (OCn/OCnX) ou sur le registre de capture (ICRn).
\( fOCnX = \frac {fclk_I/O}{2*N*(1+OCRnA)}\)
  • fOCnX Fréquence du signale générai sur OCnX.
  • fclk_I/O fréquence d'entrée du timer, (aven le pré diviseur).
  • N Valeur du pré diviseur, ( tc_clock_e ).
  • OCRnA ou ICRn valeur du comparateur.
Fast PWM.
Le(s) registre(s) de comparaison (OCn/OCnX) sont mis à jour a la valeur minimale (0) du timer. Le débordement du conteur un ci que le drapeau de débordement s'effectu sur la valeur maximale du timer.
\( RFPWM = \frac {log(TOP+1)}{log(2)}\)
  • RFPWM Résolution de la pwm générer.
  • TOP Valeur maximale du timer
    \( fOCnXPWM = \frac {fclk_I/O}{N*(1+TOP)}\)
  • fOCnXPWM Fréquence du signale générai sur OCnX.
  • fclk_I/O fréquence d'entrée du timer, (aven le pré diviseur).
  • N Valeur du pré diviseur, ( tc_clock_e ).
  • TOP Valeur maximale du timer.
PWM, Phase Correct.
Le(s) registre(s) de comparaison (OCn/OCnX) sont mis à jour à la valeur minimale (0) du timer. Le débordement du conteur s'effectu sur la valeur maximale du timer, le drapeau de débordement est lever à la valeur minimale tu timer (0)
\( RPCPWM = \frac {log(TOP+1)}{log(2)}\)
  • RPCPWM Résolution de la pwm générer.
  • TOP Valeur maximale du timer
    \( fOCnXPCPWM = \frac {fclk_I/O}{2*N*TOP}\)
  • fOCnXPCPWM Fréquence du signale générai sur OCnX.
  • fclk_I/O fréquence d'entrée du timer, (aven le pré diviseur).
  • N Valeur du pré diviseur, ( tc_clock_e ).
  • TOP Valeur maximale du timer.
PWM, Phase and Frequency Correct.
Le(s) registre(s) de comparaison (OCn/OCnX) sont mis à jour à la valeur minimale (0) du timer. Le débordement du conteur un ci que le drapeau de débordement s'effectu sur la valeur minimale (0) du timer.
\( RPFCPWM = \frac {log(TOP+1)}{log(2)}\)
  • RFPWM Résolution de la pwm générer.
  • TOP Valeur maximale du timer
    \( fOCnXPFCPWM = \frac {fclk_I/O}{2*N*TOP}\)
  • fOCnXPFCPWM Fréquence du signale générai sur OCnX.
  • fclk_I/O fréquence d'entrée du timer, (aven le pré diviseur).
  • Valeur du pré diviseur, ( tc_clock_e ).
  • TOP Valeur maximale du timer.
