static void adc_left_adjust_result ( )
inlinestatic

Pour ajuster le résulta de la conversion à gauche.

Ceci veux dire, que les 8 bits du pois le plus fore se retrouve dans l'octet du pois le plus fore du convertisseur (ADCH), les 2 bits de pois faible restent se retrouve dans les bits de de pois les plus fort de l'octet du pois le plus faible du convertisseur (ADCL).
Ceci est utiles pour les conversions sur 8 bits.
