static void tc_enable_interrupt_compare ( tc_s *  tc,
tc_compare_e  compare 
)
inlinestatic

Active l'interruption de comparaison.

Note
Pour les timers contenant qu'un seul registre de comparaison, la valeur donnée à compare n'est pas prise en compte.

A utiliser avec le vecteur d'interruption TIMERn_COMPX_vect, (n représentent le numéro du timer et X le nom du comparateur).

//Exemple pour le comparateur C du timer 4.
ISR(TIMER4_COMPC_vect)
{
//code
}
Note
Certain composent ne dispose pas des vecteurs d'interruption décri si dessus. Les autre vecteur possible sont décri si dessous.
TIM0_COMPA_vect
Le vecteur d'interruption TIM0_COMPA_vect, est utiliser par les composants suivent :
ATtiny13, ATtiny43U, ATtiny24, ATtiny44, ATtiny84, ATtiny45, ATtiny25, ATtiny85
TIM0_COMPB_vect
Le vecteur d'interruption TIM0_COMPB_vect, est utiliser par les composants suivent :
ATtiny13, ATtiny43U, ATtiny24, ATtiny44, ATtiny84, ATtiny45, ATtiny25, ATtiny85
TIM1_COMPA_vect
Le vecteur d'interruption TIM1_COMPA_vect, est utiliser par les composants suivent :
ATtiny24, ATtiny44, ATtiny84, ATtiny45, ATtiny25, ATtiny85
TIM1_COMPB_vect
Le vecteur d'interruption TIM1_COMPB_vect, est utiliser par les composants suivent :
ATtiny24, ATtiny44, ATtiny84, ATtiny45, ATtiny25, ATtiny85
TIMER1_CMPA_vect
Le vecteur d'interruption TIMER1_CMPA_vect, est utiliser par les composants suivent :
ATtiny26
TIMER1_CMPB_vect
Le vecteur d'interruption TIMER1_CMPB_vect, est utiliser par les composants suivent :
ATtiny26
TIMER1_COMP1_vect
Le vecteur d'interruption TIMER1_COMP1_vect, est utiliser par les composants suivent :

TIMER0_COMP_vect
Le vecteur d'interruption TIMER0_COMP_vect, est utiliser par les composants suivent :

TIMER1_COMP_vect
Le vecteur d'interruption TIMER1_COMP_vect, est utiliser par les composants suivent :

TIMER2_COMP_vect
Le vecteur d'interruption TIMER2_COMP_vect, est utiliser par les composants suivent :

AT90PWM3, AT90PWM2, AT90PWM1.
