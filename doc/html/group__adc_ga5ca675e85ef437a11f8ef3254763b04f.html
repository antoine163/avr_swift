enum adc_channel_e

Valeur pour sélectionner le signale d'entrée et le gain.

A utiliser avec adc_channel_select().

Note
Les "channels" avec gain sont à utiliser avec la tension de référence de 2.56V.
Voir également
adc_channel_select()
