Description détaillée

Driver i2c en maître.

Rapidement : Il faut définir la fréquence (en Hz) de fonctionnement de l'i2c via la define I2C_FREQUENCY, dans le ficher config.h. exemple ->

//********************************* Config i2c *********************************
#define I2C_FREQUENCY 400000 //400 KHz

Puis initialiser la bibliothèque via la fonction i2cInit(), activer l'i2c via i2cEnable() et vous pouvez utiliser les fonctions i2cWriteData() et i2cReadData()
